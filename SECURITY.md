# Security Policy

## Reporting a Vulnerability

If you find a security issue in this project, please report it privately:

1. Open a private security advisory on GitHub, OR
2. Email the maintainer directly.

Do not open a public issue for security vulnerabilities.

## Response Time

We aim to acknowledge reports within 48 hours and provide a fix or
mitigation within 7 days, depending on severity.

## Scope

This policy covers the security-lab-workflow framework code (scripts,
templates, skills, CI). It does not cover your local engagement data,
targets, or findings — those are your responsibility.
